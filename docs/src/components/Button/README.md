# Button

Button element

## Example

```javascript
<Button onClick={() => console.log('You clicked me!')}>Click Me!</Button>;
```

## Properties

| Property | Req | Type                  | Description                                               | Default  |
| -------- | --- | --------------------- | --------------------------------------------------------- | -------- |
| children | no  | array, object, string | Children node                                             | `Submit` |
| type     | yes | string                | Button type f.e button, submit, reset                     |          |
