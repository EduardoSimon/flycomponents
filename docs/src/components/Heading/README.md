# Heading

Wraps a header.

## Example

```javascript
<Heading as="h1" size="large" text="Title" />
```

## Properties

| Property      | Req | Type   | Description                              | Default |
| ------------- | --- | ------ | ---------------------------------------- | ------- |
| as            | no  | string | Html Tag                                 | h1      |
| size          | no  | string | Size added as class                      | huge    |
| text          | no  | string | Text show in the header                  |         |
